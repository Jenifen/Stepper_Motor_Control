## General info
This project is composed by :
* config : Board.h
* arduino_src : all packages of project
* arduino.ino : mainprogram

## Technologies
Project is created with:
### Software :
* C++
* arduino 
* LiquidCrystal_I2C library 

### Hardware :
* Arduino uno
* LCD Display 16x2
* Motor stepper 24V
* Driver stepper TB6600 4A 9 ~ 42 V
* Two potentiometers 10Kohm
* some buttons + LEDs
