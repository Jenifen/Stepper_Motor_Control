Respirator prototype [2020]

## General info
* PIN READY : 5
* PIN STOP : 2
* DIR : 6
* PUL : 7
* POT FOR PULSE PERIOD : A0
* POT FOR CYCLE PERIOD : A1
* A4, A5 : LCD 

## Technologies
Project is created with:
### Software :
* C++
* arduino 

### Hardware :
* Arduino uno
* LCD Display 16x2
* Motor stepper 24V
* Driver stepper TB6600 4A 9 ~ 42 V
* Two potentiometers 10Kohm
* some buttons + LEDs
